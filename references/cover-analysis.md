## 一、6 张封面逐张分析

### 封面 #1 — 暗色科技评测风

| 维度 | 数据 |
|------|------|
| 基调 | 暗色中灰 (mean=109)，高对比度 (std=73) |
| 主色 | 深灰 RGB(64,64,64) 44.9% + 黑灰 RGB(32,32,32) 19.6% |
| 亮色 | 白色 RGB(224,224,224) 17.9% |
| 强调色 | 红色 RGB(224,0,32) 1.1%（精准点缀） |

**爆款逻辑**：暗色背景打造专业科技感；红色作为唯一强调色制造紧迫感("新品/首发")；不对称构图让画面有层次；文字占比大，信息密度高，满足用户"看封面知内容"的浏览习惯。

## 二、爆款封面通用公式（6 张总结）

### 2.1 尺寸规范
- **540×720 像素（3:4 竖屏）** — 这是抖音封面的黄金比例
- 填满整个竖屏空间，不留黑边

### 2.2 背景策略（3 种经典方案）

| 方案 | 占比 | 主色调 | 适用场景 |
|------|------|--------|----------|
| **极致暗黑** | 4/6 | 纯黑 #000000 或深蓝黑 #000020 | 产品评测、新品首发、性能测试 |
| **暗灰科技** | 1/6 | 深灰 #404040 | 资讯解读、技术科普 |
| **明亮暖色** | 1/6 | 奶油白/米色 #E0E0C0 | 对比横评、推荐清单 |

### 2.3 色彩法则（六亲不认的铁律）

```
基础色（背景）：占据 60-80% 画面
文字色（白/黑）：占据 15-25% 画面  
强调色（红/橙/金/青）：占据 1-10% 画面（精准控制）
```

- **暗底+白字+红/金强调** = 最经典的科技爆款公式（4/6 封面使用）
- **亮底+黑字+红橙强调** = 清新可信赖的科技公式（2/6 封面使用）
- **强调色不超过 2 种**，饱和度要拉满（>80%）
- **全局对比度必须高**（std > 70），这是抓眼球的前提

### 2.4 排版铁律

```
┌──────────────────────┐
│                      │
│   🖼️ 产品图/场景图   │  ← 上部 25-40%
│   (主体偏左或居中)   │
│                      │
├──────────────────────┤
│                      │
│  🔥 钩子大标题       │  ← 中部 15-20%
│  （疑问/数字/对比）   │     字号最大
│                      │
│  副标题/规格/参数     │  ← 中下部 15-20%
│                      │
│  ¥价格 / ⭐跑分       │  ← 底部 20-25%
│  标签1  标签2  标签3  │     信息密度最高
│                      │
└──────────────────────┘
```

- **文字覆盖画面 40-60%** — 信息密度要高
- **标题用疑问句或数字开头** — "为什么…？""X款…横评""榜单TOP5"
- **底部标签化信息** — 用色块/边框包裹关键参数

### 2.5 字体层级（3-4级）

| 层级 | 用途 | 特征 |
|------|------|------|
| **L1 钩子标题** | 主标题，3-8字 | 最大号、加粗、白色/黑色、可能带描边 |
| **L2 副标题** | 补充说明，6-15字 | 中等字号、略小、不同颜色或透明度 |
| **L3 关键参数** | 价格/跑分/规格 | 醒目色块底、数字特大 |
| **L4 标签** | 话题标签 2-4个 | 小字、色块包裹、圆角边框 |

### 2.6 构图共性

1. **不对称 = 动感**：5/6 封面左右亮度明显不均
2. **中心聚焦或偏左聚焦**：产品/主体极少放右侧
3. **底部信息密度 > 顶部**：用户视线从上往下扫
4. **边缘较暗**：形成自然画框，突出中心内容

## 三、心理触发机制

| 触发点 | 实现手段 | 心理效果 |
|--------|----------|----------|
| **稀缺感** | 红色强调色、"首发""限量"标签 | FOMO 害怕错过 |
| **权威感** | 暗色背景、数字/跑分、规格罗列 | 专业可信 |
| **好奇心** | 疑问句标题、对比/VS 结构 | 信息缺口驱动点击 |
| **从众效应** | "排行""TOP""爆款"关键词 | 社会认同 |
| **视觉冲击** | 高对比度、不对称构图、饱和强调色 | 停止滑动手指 |

## 四、与普通封面的关键差异

| 维度 | 普通封面 | 爆款科技封面 |
|------|----------|-------------|
| 对比度 | 中低 | **极高** (std>70) |
| 信息密度 | 1-2 行字 | **4-8 行字**，信息轰炸 |
| 色彩数量 | 3-5 种 | **2-3 种**，极致克制 |
| 强调色 | 随便用 | **<10% 画面**，精准点缀 |
| 背景 | 场景/渐变 | **纯色/极简暗色** |
| 字体层级 | 1-2 级 | **3-4 级**清晰层级 |
| 构图 | 居中对称 | **不对称**，左重右轻 |
